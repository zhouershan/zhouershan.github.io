CF1019C Sergey's problem

关键思路：逐个满足条件。

先满足第二个条件，将编号从小到大贪心，可以做到距离不超过 111 啥的。

再满足第一个条件，选择相邻点则删去。

code 

const int N = 1e6 + 10;
vector<int> vec[N], ans;
bool vis1[N], vis2[N];
int n, m;
int main() {
    read(n, m);
    for (int u, v, i = 1; i <= m; i++) read(u, v), vec[u].push_back(v);
    for (int i = 1; i <= n; i++)
        if (!vis1[i]) { vis1[i] = vis2[i] = 1; for (int v : vec[i]) vis1[v] = 1; }
    for (int i = n; i; i--)
        if (vis2[i]) { for (int v : vec[i]) vis2[v] = 0; ans.push_back(i); }
    println(ans.size());
    for (int v : ans) write(v, ' ');
    return 0;
}

CF1386A Colors

有一个值 c<nc < nc<n，你可以用最多 646464 次询问把它查询出来。

每次询问你可以给出一个 x≤nx \le nx≤n，，假设你上次问的是 yyy，那么系统将返回 ∣x−y∣≥c|x - y| \ge c∣x−y∣≥c 是否为真。

因为第一次询问没有上次询问的值，所以系统返回什么都可以。此外，你还要保证你询问的 xxx 不重复。

n≤1018n \le 10^{18}n≤1018。

第一个思路是直接二分。

我们发现直接二分会遇到如果当前点在中间然后下一个要问 >n2> \dfrac n 2>2n​ 时就完蛋了。

因此考虑防止这种情况发生。

思路是：左右横跳。

我们考虑找到一个点 ppp 使得二分左右横跳的时候没有跳出去。

这个点可以倒着模拟最坏二分情况做来找到。

这样就可以用 log⁡V+C\log V + ClogV+C，CCC 为常数来找到答案。

    cout << "? " << x << endl;
    bool ans;
    cin >> ans;
    return ans;
}
int main() {
    int T;
    cin >> T;
    while (T--) {
        cin >> n, l = 0, r = n;
        ll pos = 0, f = 1;
        vector<ll> ans;
        while (r - l > 1) l = l + r >> 1, ans.emplace_back(l);

            pos += f * mid, f = -f;
            if (ask(pos + 1)) r = mid;
            else l = mid;
        }
        // println(r);
        cout << "= " << r << endl;
    }
    return 0;
}

[ABC254Ex] Multiply or Divide by 2

题意：

给定大小为 $ n $ 的集合 $ A $ 和 $ B $，你可以对集合 $ A $ 中的元素 $ a_i $ 进行两种操作，分别为 $ a_i \leftarrow \lfloor \dfrac{a_i}{2} \rfloor $，和 $ a_i \leftarrow a_i \times 2 $。你需要操作集合 $ A $ 直至集合 $ A, B $ 完全相同。求最小操作次数，若无解输出 -1。

n≤105n \le 10^5n≤105。

思路：

这个题很厉害啊。

我们发现：

对于 AAA 集合最大的点 xxx 和 BBB 集合最大的点 yyy：

如果 x=yx = yx=y，则说明可以消掉；

如果 x>yx > yx>y，则说明 xxx 一定要执行 /2/ 2/2 操作，试执行后重新放回集合；

如果 x<yx < yx<y，则说明 yyy 一定由 ×2\times 2×2 操作得来，执行 y/=2y /= 2y/=2 操作，试执行后重新放回集合；

然后直接模拟即可。

时间复杂度 O(nlog⁡n)O(n \log n)O(nlogn)。

code 

priority_queue<int> q1, q2;
int main() {
    int n, x;
    read(n);
    for (int i = 1; i <= n; i++) read(x), q1.push(x);
    for (int i = 1; i <= n; i++) read(x), q2.push(x);
    int ans = 0;
    while (q1.size()) {
        int a = q1.top();
        q1.pop();
        int b = q2.top();
        q2.pop();
        if (a == b) continue;
        ++ans;
        if (a > b) {
            a >>= 1;
            q1.push(a), q2.push(b);
        } else {
            if (b & 1) return println(-1), 0;
            b >>= 1;
            q1.push(a), q2.push(b);
        }
    }
    cout << ans;
    return 0;
}

P3587 [POI2015] POD

题意：

长度为 nnn 的一串项链，每颗珠子是 kkk 种颜色之一。第 iii 颗与第 i−1,i+1i-1,i+1i−1,i+1 颗珠子相邻，第 nnn 颗与第 111 颗也相邻。

切两刀，把项链断成两条链。要求每种颜色的珠子只能出现在其中一条链中。

求方案数量（保证至少存在一种），以及切成的两段长度之差绝对值的最小值。

n≤106n \le 10^6n≤106。

思路：

哈希。

环上选择可以考虑为选择一个区间。

对于每种颜色，我们在保证集合所有点的权值异或和为 000 的情况下随机赋值，那么区间成立当且仅当区间权值异或和为 000。

注意发现区间为前缀时后缀会多算一次。

CF482C Game with Strings

题意：

你和你的朋友玩一个游戏，游戏规则如下。

你的朋友创造出了 nnn 个长度均为 mmm 的不相同的字符串，然后他随机地选择其中一个。他选择这些字符串的概率是相等的，也就是说，他选择 nnn 个字符串中的每一个的概率是 1n\frac{1}{n}n1​。你想猜猜你的朋友选择了哪个字符串。

为了猜到你的朋友选择了哪个字符串，你可以问他问题，形式如下：字符串中第 pospospos 个字符是什么？当这些问题的答案能够唯一确定一个字符串时，我们认为这个字符串被猜到了。在字符串被猜到后，你将停止提问。

你没有特殊的策略，所以你每次可能会等概率的问任何一个你从没猜过的位置。求猜到你的朋友选的字符串所需次数的期望。

n≤50,len≤20n \le 50, len \le 20n≤50,len≤20。

思路：

有一个简单的 O(2lenpoly(n,len))O(2^{len}\text{poly}(n, len))O(2lenpoly(n,len)) 的做法，就是考虑状压 $\text{dp} $对于每个 iii 具体来说，设 fjf_jfj​ 表示当前已经询问了 iii 这个状态，期望再问多少次能够确定这个串（当前枚举的这个串）。

这样直接做要 O(nlen2len)O(nlen 2^{len})O(nlen2len)。

然后考虑优化：

我们记录 gig_igi​ 表示询问 iii 这个集合不能区分哪些串。

这样可以 nnn 个串一起做，时间复杂度 O(len2len)O(len 2^{len})O(len2len)。

const int N = 21, M = 55;

                g[now] |= (1ll << i - 1) | (1ll << j - 1);
            }

        int cnt = 0;
        for (int j = 0; j < m; j++)
            if (!((i >> j) & 1)) f[i] += f[i | (1 << j)], ++cnt;

    read(n);
    for (int i = 1; i <= n; i++) g[i] = i, read(fa[i]), fa[i] = i - fa[i];
    for (int i = 1; i <= n; i++)
        if (find(i) == find(fa[i])) {
            int cc = 1;
            for (int u = fa[i]; u != i; u = fa[u]) ++cc;
            write(cc, '\n', i);
            for (int u = fa[i]; u != i; u = fa[u]) write(' ', u);
            return println();
        } else
            g[find(i)] = find(fa[i]);
}
int main() {
    int T;
    read(T);
    while (T--) work();
    return 0;
}
